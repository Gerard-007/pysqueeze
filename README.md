# pysqueeze

High-quality image compression for Python, inspired by TinyPNG and Squoosh.

## Features

- Metadata stripping

## Install
```bash
pip install pysqueeze
```

# Usage
```
from pysqueeze import compress_image
compress_image("input.jpg", "output.webp")
```
```
pysqueeze input.jpg -o output.webp
```
